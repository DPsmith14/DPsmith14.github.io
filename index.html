Cubitos Probability Calculator

The colored boxes correspond to the colored dice in the game. Using the +/- buttons, select how many of each color dice you have to determine the probability you won't bust. The black die is the start player die, and you simply select whether or not you have the start player die (✔ means you have the start player die).
